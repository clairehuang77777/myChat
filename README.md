## Features
- Group chat list
- chat room
- Send messenge as chat manager (what you send will regarded as system message)
- Press reaction (Thumbs-up/Love/Laugh)
- Lazy loading
- Dark mode

## Build With
- **Frontend**: React with Vite / TailwindCSS
- **Backend**: Node.js + Express
- **Data**: JSON file
- **Deploy on**: Render
